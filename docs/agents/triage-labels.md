# Triage Labels

技能使用五个 canonical triage roles。本文件记录这些 roles 在本仓库 issue tracker 中对应的标签字符串。

| 技能 role | 本仓库标签 | 含义 |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | 需要维护者评估 |
| `needs-info` | `needs-info` | 等待报告者补充信息 |
| `ready-for-agent` | `ready-for-agent` | 规格已明确，可以交给 agent 实现 |
| `ready-for-human` | `ready-for-human` | 需要人类实现 |
| `wontfix` | `wontfix` | 不会处理 |

当技能提到某个 role 时，使用本表右侧对应的标签字符串。
