# AGENTS.md instructions

* After modifying code, always check whether there is any garbled text in the code and fix it if needed.
* The local system is Windows.
* The codebase uses UTF-8 encoding.
* Chinese characters may appear garbled in PowerShell, so always pay attention to encoding issues when viewing, editing, or generating code and text files.
* Except for code, only use Chinese to ask and answer.

## Agent skills

### Issue tracker

本仓库使用本地 markdown issue tracker，issues 和 specs 写在 `.scratch/` 下。详见 `docs/agents/issue-tracker.md`。

### Triage labels

本仓库使用默认 triage 标签词汇。详见 `docs/agents/triage-labels.md`。

本仓库使用 single-context 领域文档布局：根目录 `CONTEXT.md`，ADR 位于 `docs/adr/`。详见 `docs/agents/domain.md`。
