# Calendar MVP

This context defines the product language for the local calendar MVP.

## Language

**本地日历**:
只保存在当前设备内的日历体验，退出应用后仍保留数据。不接入账号、不同步云端、不读写系统日历。
_Avoid_: 系统日历、云日历、同步日历

**事项**:
用户在某个日期上创建的日程条目，包含标题、日期、开始时间、结束时间、备注和颜色。不包含地点、提醒、重复、参与人、附件或优先级。
_Avoid_: 任务、待办、事件

**事项时间**:
事项发生在单个日期内，按开始时间升序排列。结束时间必须晚于开始时间；不支持全天事项或跨天事项。
_Avoid_: 全天、跨天

**选中日期**:
用户在当前月份网格里点选的日期。事项列表始终显示选中日期的事项。
_Avoid_: 今天、当前日期

**当天**:
选中日期对应的那一天，用于描述列表、空状态和新增事项的归属。
_Avoid_: 今天
