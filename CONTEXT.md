# Local Calendar

This context defines the product language for the local calendar app.

## Language

**本地日历**:
只保存在当前设备内的日历体验，退出应用后仍保留数据。不接入账号、不同步云端、不读写系统日历。
_Avoid_: 系统日历、云日历、同步日历

**本地数据**:
当前设备内保存的用户数据，包括事项和待办项。不同功能页面看到的是同一份本地数据。
_Avoid_: 云数据、账号数据、页面私有数据

**事项**:
用户在某个日期上创建的日程条目，包含标题、日期、开始时间、结束时间、备注和颜色。不包含地点、提醒、重复、参与人、附件或优先级。
_Avoid_: 任务、待办、事件

**事项时间**:
事项发生在单个日期内，按开始时间升序排列。结束时间必须晚于开始时间；不支持全天事项或跨天事项。
_Avoid_: 全天、跨天

**选中日期**:
用户在当前显示月份网格里点选的日期。事项列表始终显示选中日期的事项。
_Avoid_: 今天、当前日期

**当前显示月份**:
日历页正在展示的月份，用户可以切换到上个月或下个月。当前显示月份不是固定的演示月份。
_Avoid_: 固定月份、演示月份

**月份切换**:
用户在日历页切换到上个月或下个月的操作。月份切换后优先选中目标月份中相同日序的日期；如果目标月份没有该日序，则选中目标月份最后一天。月份切换不表示年份/月选择器，也不表示跳转今天。
_Avoid_: 年份选择、月份选择器、跳转今天

**当天**:
选中日期对应的那一天，用于描述列表、空状态和新增事项的归属。
_Avoid_: 今天

**功能页面**:
底部导航中用户可以进入的一级使用区域，包括日历页、日程页、待办页和我的页。功能页面不是静态装饰入口。
_Avoid_: 视觉壳、占位入口

**功能页面状态**:
用户在功能页面内形成的当前查看位置或选择，例如日历页的选中日期。切换底部导航时应保留功能页面状态。
_Avoid_: 临时页面、重置页面

**日历页**:
围绕月份网格和选中日期查看当天事项的功能页面。
_Avoid_: 首页、仪表盘

**日程页**:
跨日期查看和删除已有事项的功能页面，面向事项的列表阅读和查找。日程页显示全部本地事项，按日期从早到晚分组，组内按事项时间升序排列；没有事项时引导用户去日历页添加，日程页不是新增事项的入口。
_Avoid_: 待办页、任务页、新增事项页

**待办页**:
管理不一定绑定事项时间的轻量清单的功能页面。待办页是待办项的主创建入口，支持新增、完成切换和删除待办项；没有待办项时引导用户新增待办，未完成待办项显示在已完成待办项之前。
_Avoid_: 事项页、日程页

**待办项**:
用户在待办页创建的轻量清单条目，包含标题、是否完成、创建时间和可选备注。待办项不包含截止日期、优先级、分类、提醒、重复或事项时间；删除待办项前必须二次确认。
_Avoid_: 事项、任务、日程

**我的页**:
管理本地应用信息、本地数据统计和本地数据清理入口的功能页面。我的页不表示账号、云同步、会员或备份。
_Avoid_: 账号页、个人中心、云同步页

**清空本地数据**:
用户主动删除当前设备内全部事项和待办项的操作。清空前必须二次确认；清空后保持当前功能页面和功能页面状态不变，数据相关区域自然显示空状态或零统计。
_Avoid_: 退出登录、云端删除、重置账号
