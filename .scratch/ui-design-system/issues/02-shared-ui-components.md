# 02 — 共享 UI 组件首批落地

**What to build:** 提供首批跨页面共享 UI 组件，让四个功能页面能复用一致的页面容器、页面标题、内容卡片、空状态、统计行、确认弹窗和底部表单外壳。完成后至少有一条真实用户路径使用这些组件，证明共享组件能承载现有体验。

**Blocked by:** 01 — 应用级主题 token 骨架

**Status:** resolved

- [x] 共享页面容器统一页面宽度、背景承载、安全区和底部导航避让。
- [x] 共享页面标题统一标题、说明文字和页面顶部间距。
- [x] 共享内容卡片统一表面色、边框、圆角、阴影和内边距。
- [x] 共享空状态统一图标、标题、说明和主操作布局。
- [x] 共享统计行或统计块统一标签、数字和层级。
- [x] 共享确认弹窗统一风险操作标题、正文、取消按钮和确认按钮样式。
- [x] 共享底部表单外壳统一标题、字段区域、操作按钮和底部安全区处理。
- [x] 至少迁移一个真实页面区域或操作路径到共享组件。
- [x] 相关 widget 测试覆盖共享组件在真实路径中的可见行为。
