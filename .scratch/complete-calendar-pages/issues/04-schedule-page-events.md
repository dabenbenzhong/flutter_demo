# 04 — 日程页查看和删除事项

**What to build:** 用户可以在日程页跨日期查看全部本地事项。事项按日期从早到晚分组，组内按事项时间升序排列；用户可以在日程页删除事项，删除前需要二次确认。

**Blocked by:** 02 — 四功能页面导航骨架

**Status:** ready-for-agent

- [ ] 日程页显示全部本地事项。
- [ ] 事项按日期从早到晚分组。
- [ ] 同一日期内事项按事项时间升序排列。
- [ ] 日程页不提供新增事项表单。
- [ ] 日程页没有事项时显示“还没有事项”或等价空状态。
- [ ] 日程页空状态提供去日历页添加的入口。
- [ ] 点击空状态入口后切换到日历页，但不自动打开新增事项表单。
- [ ] 日程页每个事项提供删除入口。
- [ ] 点击删除入口后显示二次确认弹窗。
- [ ] 取消删除后事项保持不变。
- [ ] 确认删除后事项从日程页消失。
- [ ] 日程页删除事项后，日历页当天列表和日期标记同步更新。
- [ ] 相关 widget 测试覆盖分组排序、空状态跳转、删除取消和删除确认。
