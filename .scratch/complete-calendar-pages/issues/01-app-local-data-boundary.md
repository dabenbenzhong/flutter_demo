# 01 — 应用级本地数据边界

**What to build:** 将现有事项存储扩展为应用级本地数据边界，让应用能用同一份本地数据读取、保存、统计和清空事项与待办项。完成后用户数据仍只保存在当前设备，关闭并重新打开应用后不会丢失。

**Blocked by:** None — can start immediately.

**Status:** ready-for-agent

- [ ] 本地数据边界同时支持事项和待办项。
- [ ] 待办项包含标题、是否完成、创建时间和可选备注。
- [ ] 事项数据的现有持久化行为保持不变。
- [ ] 待办项可以通过内存存储和文件存储保存并重新加载。
- [ ] 本地数据边界能返回事项数量和待办项数量。
- [ ] 本地数据边界能清空全部事项和待办项。
- [ ] 文件存储使用轻量 JSON/键值边界，不引入数据库。
- [ ] 相关存储测试覆盖事项和待办项 round-trip、统计和清空。
