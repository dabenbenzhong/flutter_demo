# 07 — 我的页统计和清空本地数据

**What to build:** 用户可以在我的页查看应用信息、本地事项数量和本地待办项数量，也可以二次确认后清空全部本地数据。清空后保持当前功能页面和功能页面状态不变，各数据区域自然显示空状态或零统计。

**Blocked by:** 04 — 日程页查看和删除事项；06 — 待办页完成状态和删除确认

**Status:** ready-for-agent

- [ ] 我的页显示应用名称“日历”。
- [ ] 我的页显示本地应用说明，不出现账号、云同步、会员或备份语义。
- [ ] 我的页显示事项数量。
- [ ] 我的页显示待办项数量。
- [ ] 新增或删除事项后，我的页事项数量同步更新。
- [ ] 新增或删除待办项后，我的页待办项数量同步更新。
- [ ] 我的页提供清空本地数据入口。
- [ ] 点击清空入口后显示二次确认弹窗。
- [ ] 取消清空后全部事项和待办项保持不变。
- [ ] 确认清空后全部事项和待办项被删除。
- [ ] 清空后日程页显示空状态。
- [ ] 清空后待办页显示空状态。
- [ ] 清空后我的页事项数量和待办项数量显示为 0。
- [ ] 清空后保持当前功能页面不变。
- [ ] 清空后日历页当前显示月份和选中日期保持不变。
- [ ] 重新打开应用后，被清空的数据不会重新出现。
- [ ] 相关 widget 测试覆盖统计、清空取消、清空确认和清空后页面状态。
