# 05 — 待办页新增和持久化待办项

**What to build:** 用户可以在待办页创建待办项。待办项包含标题、是否完成、创建时间和可选备注；标题必填，备注可选；新增后立即显示在待办页，并在重新打开应用后仍然存在。

**Blocked by:** 01 — 应用级本地数据边界；02 — 四功能页面导航骨架

**Status:** ready-for-agent

- [ ] 待办页没有待办项时显示空状态。
- [ ] 待办页空状态提供新增待办入口。
- [ ] 待办页有新增待办入口。
- [ ] 新增待办表单包含标题输入。
- [ ] 新增待办表单包含可选备注输入。
- [ ] 标题为空时不能保存，并显示校验反馈。
- [ ] 保存有效待办项后关闭表单。
- [ ] 新待办项立即出现在待办页。
- [ ] 新待办项默认是未完成状态。
- [ ] 新待办项保存创建时间。
- [ ] 关闭并重新打开应用后，已创建待办项仍显示。
- [ ] 相关 widget 测试覆盖空状态、新增、标题校验和持久化加载。
