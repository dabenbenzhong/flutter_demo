# 03 — 日历页月份切换

**What to build:** 用户可以在日历页切换到上个月或下个月。当前显示月份、月份网格、选中日期、当天事项列表、新增事项归属和日期标记都跟随月份切换。

**Blocked by:** 02 — 四功能页面导航骨架

**Status:** resolved

- [x] 日历页显示当前显示月份，而不是固定演示月份。
- [x] 用户可以切换到上个月。
- [x] 用户可以切换到下个月。
- [x] 月份切换后优先选中目标月份相同日序。
- [x] 当目标月份没有原日序时，选中目标月份最后一天。
- [x] 月份网格显示当前显示月份的真实日期。
- [x] 新增事项使用月份切换后的当前选中日期。
- [x] 日期标记只反映当前显示月份内有事项的日期。
- [x] 切换 tab 后再回到日历页，当前显示月份和选中日期保持不变。
- [x] 相关 widget 测试覆盖上个月、下个月、月末回退和跨月份事项创建。
