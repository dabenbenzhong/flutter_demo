# 04 — 删除事项并二次确认

**What to build:** 用户可以从当天事项列表删除事项；删除前弹出确认框，取消后事项保持不变，确认后事项从选中日期列表消失。

**Blocked by:** 02 — 通过底部表单创建事项

**Status:** resolved

- [x] 每个事项提供清晰的删除入口。
- [x] 点击删除入口后显示二次确认弹窗。
- [x] 取消确认后，事项仍保留在当天事项列表中。
- [x] 确认删除后，事项从当天事项列表中消失。
- [x] 删除某一天的事项不会影响其他日期的事项。
- [x] 删除最后一个事项后，选中日期显示“当天没有事项”。
- [x] 相关 widget 测试覆盖删除取消路径和确认路径。
