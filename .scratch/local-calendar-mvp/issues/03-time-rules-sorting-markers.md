# 03 — 补齐事项时间规则、排序和日期标记

**What to build:** 用户创建事项时，应用阻止结束时间不晚于开始时间的无效事项；当天事项按开始时间升序排列；月历网格标记有事项的日期，让用户能快速扫出哪些天有安排。

**Blocked by:** 02 — 通过底部表单创建事项

**Status:** resolved

- [x] 结束时间不晚于开始时间时不能保存事项。
- [x] 无效事项时间会显示清晰的校验反馈。
- [x] 同一选中日期的事项按开始时间升序显示。
- [x] 新增多个不同时间事项后，列表顺序仍稳定。
- [x] 有事项的日期在月历网格中显示标记。
- [x] 删除或切换日期前后，日期标记与真实事项状态保持一致。
- [x] 相关 widget 测试覆盖无效时间校验、升序排序和日期标记。
