# 01 — 让选中日期驱动当天事项列表

**What to build:** 用户点击当前月份网格中的日期后，选中态移动，事项区域切换到该选中日期；没有事项时显示“当天没有事项”。同时把静态日历页面调整为由选中日期和事项状态驱动，为后续新增、删除和持久化铺好最小接口。

**Blocked by:** None — can start immediately.
