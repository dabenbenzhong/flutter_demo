# 03 — 待办页编辑待办项

**What to build:** 用户可以从待办页打开预填的编辑表单，修改已有待办项的标题和备注；保存后原待办项被替换并持久化，同时保留原完成状态和创建时间，取消时原待办项保持不变。

**Blocked by:** None — can start immediately

**Status:** resolved

- [x] 待办页每条待办项提供明确可见的编辑入口，且不依赖长按或点击整行。
- [x] 编辑入口具有包含待办项语义的可访问名称。
- [x] 编辑表单标题和提交语义明确表示正在编辑待办项。
- [x] 编辑表单预填原待办项的标题和备注。
- [x] 用户可以修改、补充或清空备注。
- [x] 待办项标题继续执行新增待办项时的必填校验。
- [x] 用户修正标题后错误提示及时消失，未触碰字段不会在提交前提前显示错误。
- [x] 保存后替换原待办项，不新增副本，待办项总数保持不变。
- [x] 编辑完成的待办项后保持完成状态，编辑未完成的待办项后保持未完成状态。
- [x] 编辑后保留原创建时间，同组排序语义不因编辑而改变。
- [x] 保存后编辑结果写入本地数据，重新构建或重新加载应用后仍然存在。
- [x] 取消或关闭编辑表单后，原待办项及本地数据保持不变。
- [x] 原有新增、完成切换、取消完成和删除确认行为继续正常工作。
- [x] “我的”页待办项统计在编辑前后保持不变。
- [x] 相关用户可观察行为由最高层应用 widget 测试覆盖。

## Answer

已实现待办项编辑，并验证状态、创建时间、统计、取消与重建持久化。
